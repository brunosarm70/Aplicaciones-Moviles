....:::: INSTRUCCIONES PARA COMPILAR Y CORRER EL PROGRAMA ::::....

1. Utilizar arquitectura x86, debido a que el DLL fue creado en esta.

2. Se debe tener instalado el JDK 8.0.

3. Las 3 clases ya están compiladas (Calculator, CalculatorUI y Functions), pero en caso de querer volver a hacerlo, se utiliza el comando "javac" (ejemplo: javac Functions.java) esto generara un archivo .class.

4. La librería de las funciones en C para la calculadora (Functions.dll), ya se encuentra en la carpeta. Además del .c y el .h con los cuales se generó dicha librería. 

5. Para ejecutar el programa, se debe usar el comando "java" de la siguiente forma (java CalculatorUI).
