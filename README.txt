COMO COMPILAR y CORRERLO:

1. Utilizar arquitectura x86, debido a que el DLL fue creado en esta.

2. Las 3 clases ya vienen compiladas, (Calculator, CalculatorUI y Functions), pero en caso de querer volver a hacerlo
   utilizar el comando "javac" espacio y el nombre del .java (ejemplo: javac Functions.java) esto generara los .class .

3. Para correrlo, SOLO correr el archivo "CalculatorUI" de la siguiente forma: java CalculatorUI 
   y el programa se ejecutara.
